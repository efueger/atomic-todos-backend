# Atomic Todos: The backend

#### Tech stack:
- Node.js 5.8
- Express

#### Pre config:
- `ATOMIC_MONGO_URL`: URL to the mongo database.

#### Running:
 - `npm install`
 - `npm start`

#### Running with hot reload using nodemon:
- `npm run dev`

### Testing:
###### All `npm test`
###### Unit `npm run test-unit`
###### Integration `npm run test-integration`
